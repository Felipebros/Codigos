# uv com Python

## Instalando o uv

https://docs.astral.sh/uv/

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

## Para iniciar o uv
Dentro da pasta do projeto
```bash
uv init
```

## Como adiconar o Django por exemplo
```bash
uv add Django==5.1
```

## Instalar bibliotecas de desenvolvimento
```bash
uv add --dev ruff
```
```bash
uv add --dev ipython
```
```bash
uv add --dev ipdb
```
